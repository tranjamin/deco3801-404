# Demo

## Google Chrome

1. Open Google Chrome.
2. Go to `chrome://extensions/`.
3. Click `service worker`.
4. Open Console.
5. Open a new tab.
6. Search `google.com`.
7. Check Console.

## Microsoft Edge

1. Open Microsoft Edge.
2. Go to `edge://extensions/`.
3. Click `service worker`.
4. Open Console.
5. Open a new tab.
6. Search `google.com`.
7. Check Console.
